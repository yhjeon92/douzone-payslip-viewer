# 더존비즈온 급여명세서 뷰어

### Prerequisites

- Python 3
- PIP 3

### Getting Started

- 필수 Python 패키지 설치

```shell
$ pip3 install -r requirements.txt
```

- 서버 프로세스 시작

```shell
$ python3 server.py
```

- 복호화 스크립트를 이용해 바로 복호화된 `decoded.html` 파일 생성하기

```shell
$ python3 decipher.py my_payslip.html my_password
```
